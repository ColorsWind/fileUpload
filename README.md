## 项目初衷
希望可以帮到和我一样在收集文件时头晕眼花的朋友。
本产品致力于规范命名提交文件，为收集者提供管理，批量打包下载等功能，为用户只见提供留言功能。

## 环境要求
* macOS or Linux or Windows
* JDK1.8
* Mysql
## 项目技术栈
* 后端 : SpringBoot SpringMVC Mybatis
* 前端 : Bootstrap jQuery

## pom 主要依赖
* thymeleaf
* fastjson
* lombok
